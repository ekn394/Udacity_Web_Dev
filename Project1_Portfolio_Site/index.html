Retro Controller Hardware Hack

All you need to convert your favorite retro game controller into a plug-and-play USB controller is some scrap wires, some solder, an afternoon, and an inexpensive micro controller (in this case a Teensy++ 2.0).

This activity taught kids how to make their own video games using "Scratch". Our goal was to increase computer literacy in today's youth. 3 Googlers, and 3 tech savvy science teachers from across Canada were tapped to teach at this event. (Photos: Tamara Pimentel/CBC)
